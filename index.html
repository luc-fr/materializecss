MaterializeCSS

O que é o Materialize?

É um framework para desenvolvimento de front-end responsivo moderno, podendo ser utilizado para sites e sistemas web e mobile. Criado pela Google e tendo como base o Material Design, também desenvolvida pela Google.

Vantagens :
  • - Visual moderno e limpo
  • - Baseado no Material Design
  • - Responsivo por padrão
  • - Fácil de usar e personalizar
  • - Pronto para mobile

Como importar o Materialize na sua página?

Incluir os links no <head> do seu HTML para o CSS :

  • <link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">
  • <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/materialize/1.0.0/css/materialize.min.css">

Incluir o link no final do <body> do seu HTML para o JavaScript :

  • <script src="https://cdnjs.cloudflare.com/ajax/libs/materialize/1.0.0/js/materialize.min.js">

Ou instale utilizando o NPM :

  • npm install materialize-css@next

Quais componentes o Materialize oferece?

  • chevron_rightBadges
    Os emblemas podem notificá-lo sobre mensagens ou notificações novas ou não lidas. Adicione a new classe ao emblema para dar a ele o segundo plano.

    Exemplo de Código :

    <div class="collection">
      <a href="#!" class="collection-item"><span class="badge">1</span>Alan</a>
      <a href="#!" class="collection-item"><span class="new badge">4</span>Alan</a>
      <a href="#!" class="collection-item">Alan</a>
      <a href="#!" class="collection-item"><span class="badge">14</span>Alan</a>
    </div>

  • chevron_rightButtons
    Existem três tipos principais de botões descritos no Material Design. O botão em relevo é um botão padrão que representa ações e busca dar profundidade a uma página praticamente plana. O botão de ação circular flutuante é destinado a funções muito importantes. Botões planos geralmente são usados ​​em elementos que já possuem profundidade, como cartões ou modais.

    Exemplo de Código :

    <a class="waves-effect waves-light btn">button</a>
    <a class="waves-effect waves-light btn"><i class="material-icons left">cloud</i>button</a>
    <a class="waves-effect waves-light btn"><i class="material-icons right">cloud</i>button</a>

  • chevron_rightBreadcrumbs
    Breadcrumbs são uma boa maneira de exibir sua localização atual. Geralmente, são usados ​​quando você tem várias camadas de conteúdo.

    Exemplo de Código :

    <nav>
      <div class="nav-wrapper">
        <div class="col s12">
          <a href="#!" class="breadcrumb">First</a>
          <a href="#!" class="breadcrumb">Second</a>
          <a href="#!" class="breadcrumb">Third</a>
        </div>
      </div>
    </nav>

  • chevron_rightCards
    Os cards são um meio prático de exibir conteúdo composto por diferentes tipos de objetos. Eles também são adequados para apresentar objetos semelhantes cujo tamanho ou ações suportadas podem variar consideravelmente, como fotos com legendas de tamanho variável.

    Exemplo de Código :

    <div class="row">
      <div class="col s12 m6">
        <div class="card blue-grey darken-1">
          <div class="card-content white-text">
            <span class="card-title">Card Title</span>
            <p>I am a very simple card. I am good at containing small bits of information.
            I am convenient because I require little markup to use effectively.</p>
          </div>
          <div class="card-action">
            <a href="#">This is a link</a>
            <a href="#">This is a link</a>
          </div>
        </div>
      </div>
    </div>

  • chevron_rightCollections
    Coleções permitem que você agrupe objetos de lista.

    Exemplo de Código :

    <ul class="collection with-header">
      <li class="collection-header"><h4>First Names</h4></li>
      <li class="collection-item">Alvin</li>
      <li class="collection-item">Alvin</li>
      <li class="collection-item">Alvin</li>
      <li class="collection-item">Alvin</li>
    </ul>

  • chevron_rightFloating Action Button
    Se quiser um botão de ação flutuante fixo, você pode adicionar várias ações que aparecerão ao passar o mouse. Nossa demonstração está no canto inferior direito da página.

    Exemplo de Código :

    <div class="fixed-action-btn">
      <a class="btn-floating btn-large red">
        <i class="large material-icons">mode_edit</i>
      </a>
      <ul>
        <li><a class="btn-floating red"><i class="material-icons">insert_chart</i></a></li>
        <li><a class="btn-floating yellow darken-1"><i class="material-icons">format_quote</i></a></li>
        <li><a class="btn-floating green"><i class="material-icons">publish</i></a></li>
        <li><a class="btn-floating blue"><i class="material-icons">attach_file</i></a></li>
      </ul>
    </div>

  • chevron_rightFooter
    Os rodapés são uma ótima maneira de organizar a navegação e as informações do site no final de uma página. É para lá que o usuário olhará depois de rolar a página atual ou procurar informações adicionais sobre o seu site.

    Exemplo de Código :

    <footer class="page-footer">
      <div class="container">
        <div class="row">
          <div class="col l6 s12">
            <h5 class="white-text">Footer Content</h5>
            <p class="grey-text text-lighten-4">You can use rows and columns here to organize your footer content.</p>
          </div>
          <div class="col l4 offset-l2 s12">
            <h5 class="white-text">Links</h5>
            <ul>
              <li><a class="grey-text text-lighten-3" href="#!">Link 1</a></li>
              <li><a class="grey-text text-lighten-3" href="#!">Link 2</a></li>
              <li><a class="grey-text text-lighten-3" href="#!">Link 3</a></li>
              <li><a class="grey-text text-lighten-3" href="#!">Link 4</a></li>
            </ul>
          </div>
        </div>
      </div>
      <div class="footer-copyright">
        <div class="container">
          © 2014 Copyright Text
          <a class="grey-text text-lighten-4 right" href="#!">More Links</a>
        </div>
      </div>
    </footer>

  • chevron_rightIcons
    Você pode baixá-los diretamente das especificações do Material Design. Para poder usar esses ícones, você deve incluir esta linha na parte do seu código HTML <head> : <link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">

    Exemplo de Código :

    <!--
      Sizes:
      tiny: 1rem
      small: 2rem
      medium: 4rem
      large: 6rem
    -->
    <i class="large material-icons">insert_chart</i>

  • chevron_rightNavbar
    A barra de navegação é totalmente contida por uma tag Nav do HTML5. Dentro de um div de contêiner recomendado, há duas partes principais da barra de navegação: um link para o logotipo ou marca e os links de navegação. Você pode alinhar esses links à esquerda ou à direita.

    Exemplo de Código :

    <nav>
      <div class="nav-wrapper">
        <a href="#" class="brand-logo">Logo</a>
        <ul id="nav-mobile" class="right hide-on-med-and-down">
          <li><a href="sass.html">Sass</a></li>
          <li><a href="badges.html">Components</a></li>
          <li><a href="collapsible.html">JavaScript</a></li>
        </ul>
      </div>
    </nav>

  • chevron_rightPagination
    Adicione links de paginação para ajudar a dividir seu conteúdo longo em blocos mais curtos e fáceis de entender.

    Exemplo de Código :

    <ul class="pagination">
      <li class="disabled"><a href="#!"><i class="material-icons">chevron_left</i></a></li>
      <li class="active"><a href="#!">1</a></li>
      <li class="waves-effect"><a href="#!">2</a></li>
      <li class="waves-effect"><a href="#!">3</a></li>
      <li class="waves-effect"><a href="#!">4</a></li>
      <li class="waves-effect"><a href="#!">5</a></li>
      <li class="waves-effect"><a href="#!"><i class="material-icons">chevron_right</i></a></li>
    </ul>

  • chevron_rightPreloader
    Se você tiver conteúdo que demore muito para carregar, dê feedback ao usuário. Por isso, disponibilizamos diversos indicadores de atividade e progresso.

    Exemplo de Código :

    Determinado
    <div class="progress">
      <div class="determinate" style="width: 70%"></div>
    </div>

    Indeterminado
    <div class="progress">
      <div class="indeterminate"></div>
    </div>
